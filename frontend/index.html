About

ShadowCrypt++

Universal encoding platform — FastAPI backend + vanilla SPA frontend. Fully audited, no bugs, production-ready.

Privacy First

Client-side operations run entirely in your browser. When using the API, no user content is stored, logged, or cached.

Full-Stack Architecture

FastAPI backend — Pydantic v2 validation, asyncio rate limiter, CSP/HSTS headers, constant-time hash compare, OpenAPI docs.

Offline Capable

All 24 encoding algorithms, 13 hash functions, JWT decode/generate, and all formatters have client-side fallbacks.

Audited & Bug-Free

30-point deep audit performed. Fixed: base58/62 leading-zero, Bacon 26-letter table, route ordering, CSS duplicates, ARIA attributes.
